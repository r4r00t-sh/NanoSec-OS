# Welcome to NanoSec OS Wiki

NanoSec OS is a security-focused hobby operating system written from scratch in C and x86 Assembly.

## Features Overview

| Category | Features |
|----------|----------|
| **Core** | Preemptive multitasking, virtual memory, system calls |
| **Shell** | Pipes, redirects, 50+ Unix commands |
| **Scripting** | Nash - custom scripting language |
| **Networking** | Full TCP/IP stack, firewall |
| **Security** | ASLR, stack protection, audit logging |

## Getting Started

```bash
# Quick build and run
make clean && make full && make run-iso

# Login
Username: root
Password: root
```
